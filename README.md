# 🔐 Admin Dashboard – Next.js 16 + Shadcn/ui + Prisma

A modern, fully responsive admin dashboard with complete user management system.  
Built with **Next.js 16 (App Router)**, **Tailwind CSS**, **Shadcn/ui**, **Prisma ORM** and **SQLite**.

---

## ✨ Features

- 🔑 **Admin‑only authentication** (role‑based access)
- 📊 **Dashboard overview** with KPI cards & quick stats
- 👥 **User management** – Create, Edit, Delete (single & bulk)
- ✏️ **Bulk edit** users (role & status)
- 📱 **Fully responsive** – seamless on mobile, tablet & desktop
- 🌗 **Dark / Light mode** (built‑in with Shadcn/ui)
- 🧩 **Clean & modern UI** – powered by Shadcn/ui and Tailwind CSS
- 🗄️ **SQLite database** – easy setup, no external DB server required
- 🚀 **Optimized API routes** for all user operations

---

## 🛠️ Tech Stack

- [Next.js 16](https://nextjs.org/) – React framework (App Router)
- [TypeScript](https://www.typescriptlang.org/) – type safety
- [Tailwind CSS](https://tailwindcss.com/) – utility‑first styling
- [Shadcn/ui](https://ui.shadcn.com/) – high‑quality UI components
- [Prisma ORM](https://www.prisma.io/) – database management
- [SQLite](https://www.sqlite.org/) – lightweight database
- [Lucide React](https://lucide.dev/) – beautiful icons
